# MOTIS PHP Client

A **PHP Composer package** for the [MOTIS API](https://motis-project.de/), enabling multimodal routing, geocoding, and map tiles in PHP applications. Built with [OpenAPI Generator](https://openapi-generator.tech/), this package provides type-safe models, PSR-18 HTTP client support, and a modular architecture for easy integration with **Laravel**, **Symfony**, and **Nextcloud**.

---

## Features (planned)
- **OpenAPI-generated**: Always up-to-date with the MOTIS API.
- **PSR-18 compatible**: Works with any PSR-18 HTTP client (e.g., Guzzle).
- **Modular**: Extensible for framework-specific adapters.
- **Tested**: Includes PHPUnit tests for core functionality.

---

## Installation
```bash
composer require sneakyx/motis-php-client
```
